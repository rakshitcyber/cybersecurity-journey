# 🔐 Secure Password Generator

A simple Python program that generates strong and random passwords using letters, numbers, and optional special characters.

---

## 📌 Features

- Generate one or more passwords
- User can choose the password length
- Option to include or exclude special characters
- Ensures a minimum password length of 8 characters
- Easy to use and beginner-friendly

---

## 🛠️ Technologies Used

- Python 3
- random module
- string module

---

## 📂 Project Structure

```
Password-Generator/
│── password_generator.py
└── README.md
```

---

## ▶️ How to Run

1. Make sure Python 3 is installed.
2. Open a terminal in the project folder.
3. Run:

```bash
python password_generator.py
```
